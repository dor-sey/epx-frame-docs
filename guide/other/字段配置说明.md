# 字段配置说明

## 1. 模型概述

字段配置模型定义了表单中每个字段的完整行为特征。该模型采用**声明式JSON配置**，通过**数据类型驱动**和**条件化显示**，实现字段从基础属性到高级业务逻辑的全面控制。模型支持**100+配置项**，涵盖字段显示、验证、交互、数据绑定等全场景需求。

## 2. 核心配置结构

每个字段配置项遵循统一结构：

```javascript
{
  "key": "configKey",             // 配置项标识符
  "attriName": "字段编号",        // 字段编号
  "label": "配置项标签",         // 界面显示标签
  "type": "fieldType",           // 表单控件类型
  "defaultValue": "默认值",      // 默认值（可选）
  "readonly": true/false,        // 是否只读（可选）
  "note": "说明文字",            // 字段说明
  // 类型特定字段...
}
```

## 3. 配置项分类详解

### 3.1 基础属性配置
定义字段的核心元数据：

| 配置项 | 类型 | 描述 | 特性 |
|--------|------|------|------|
| **attriName** | text | 字段标识符 | `readonly: true` 自动生成 |
| **alias** | text | 自定义编号 | 自定义字段标识符 |
| **attriLabel** | text | 字段显示名称 | 界面显示的标签文本 |
| **placeholder** | text | 输入框提示 | 占位符文本 |

### 3.2 状态与行为控制
控制字段在不同场景下的交互状态：

| 配置项 | 类型 | 描述 | 选项 |
|--------|------|------|------|
| **state** | radio | 字段状态 | 跟随页面状态/新增/编辑/详情/只读/隐藏 |
| **isRequired** | radio | 是否必填 | 是(`1`)/否(`0`) |
| **submitValue** | switch | 提交字段值 | 控制是否参与数据提交 |
| **disabled** | switch | 禁用字段 | 只读样式展示选项 |
| **showLabel** | radio | 显示名称 | 显示(`true`)/不显示(`false`)标签 |

### 3.3 布局与样式配置
控制字段的视觉表现和布局：

| 配置项 | 类型 | 描述 | 条件 |
|--------|------|------|------|
| **colspan** | num | 占用列数 | 网格布局中的列跨度 |
| **color** | color | 字体颜色 | 默认`#333` |
| **noteMode** | radio | 注释显示模式 | 弹窗(`tips`)/文本(`text`) |
| **notes** | textarea | 注释说明 | 字段说明文本 |
| **noteTipIconColor** | color | 注释图标颜色 | `noteMode`为`tips`时显示 |

### 3.4 数据类型特定配置
根据不同数据类型显示对应的配置项：

#### 文本类型字段 (`text`, `textarea`, `bigtext`)
| 配置项 | 类型 | 描述 | 适用类型 |
|--------|------|------|----------|
| **maxLength** | num | 输入长度限制 | `text`, `textarea`, `bigtext` |
| **rows** | num | 显示行数 | `textarea`, `bigtext` |
| **limitRows** | num | 输入行数限制 | `textarea`, `bigtext` |
| **fieldExMethod** | radio | 字段扩展方式 | `text`, `num` |
| **suffix** | text | 前/后缀内容 | `text`, `num` |

#### 数字类型字段 (`num`)
| 配置项 | 类型 | 描述 | 备注 |
|--------|------|------|------|
| **numericalUnits** | radio | 数值单位 | 从代码表`Numerical units`获取 |
| **showWay** | radio | 显示内容 | 字段值(`value`)/统计值(`statistics`) |
| **statisticsType** | radio | 统计方式 | 从`FIELD_STATISTIC_TYPE`获取 |

#### 日期时间字段 (`date`, `time`, `date_ym`)
| 配置项 | 类型 | 描述 | 功能 |
|--------|------|------|------|
| **isDateNow** | radio | 是否当前时间 | 自动填充当前时间 |
| **selectRange** | dateSelectRange | 选择范围 | 日期范围限制 |

#### 对象类型字段 (`object`, `multiobject`, `user`, `dept`)
| 配置项 | 类型 | 描述 | 条件 |
|--------|------|------|------|
| **detailPageId** | object | 查看页面 | 从`FM_PAGE_CONFIG`选择 |
| **addBtn** | radio | 新增按钮 | 是否需要新增功能 |
| **addPageId** | object | 新增页面 | `addBtn: true`时显示 |
| **findPageId** | object | 对象查找页面 | 对象选择页面 |
| **objectScope** | object-scope | 对象查询范围 | 数据筛选条件 |
| **expand** | radio | 展开选项 | 是否展开树形选项 |
| **linkField** | radio | 获取关联字段 | 是否获取关联字段值 |
| **onlySelectLeaf** | radio | 只选择叶节点 | 树形选择限制 |

#### 列表选择字段 (`list`, `treeList`, `multibox`)
| 配置项 | 类型 | 描述 | 功能 |
|--------|------|------|------|
| **showOptionCodes** | option-scope | 选项显示范围 | 限制显示哪些选项 |
| **showOptionCode** | switch | 显示选项编号 | 是否显示选项代码 |
| **showLevel** | number | 下拉框层级 | 树形显示层级深度 |
| **valueOrder** | switch | 数据排序 | 是否对选项排序 |

#### 文件字段 (`blob`, `multiblob`)
| 配置项 | 类型 | 描述 | 默认值 |
|--------|------|------|--------|
| **uploadText** | text | 上传按钮文字 | 自定义按钮文本 |
| **hiddenSuffix** | switch | 隐藏文件后缀 | 隐藏文件扩展名 |
| **previewDialogSize** | radio | 预览弹窗大小 | 50%/75%/80%/90%/全屏 |

#### 条件范围字段 (`scope`)
| 配置项 | 类型 | 描述 | 作用 |
|--------|------|------|------|
| **scopeEntity** | scopeEntity | 查询对象 | 指定查询的数据实体 |

#### 业务规则字段 (`rule`)
| 配置项 | 类型 | 描述 | 功能 |
|--------|------|------|------|
| **metaId** | object | 业务规则对象 | 从`FM_ENTITY`选择规则实体 |

### 3.5 高级功能配置
实现复杂的业务交互：

#### 链接功能
| 配置项 | 类型 | 描述 | 条件链 |
|--------|------|------|--------|
| **link** | switch | 启用链接 | 文本/对象类型字段 |
| **linkType** | radio | 链接类型 | 指定页面(`page`)/URL(`url`) |
| **linkPageId** | object | 链接页面 | `linkType: 'page'`时显示 |
| **linkAddress** | text | 链接地址 | `linkType: 'url'`时显示 |
| **linkOpenType** | radio | 打开方式 | 从`OPEN_PAGE_MODE`获取 |
| **linkParams** | params | 链接参数 | 页面跳转参数 |

#### 关联字段配置
| 配置项 | 类型 | 描述 | 条件 |
|--------|------|------|------|
| **linkFieldFlag** | object | 关联标识符 | 从`FM_ENTITY_RELATION`选择 |
| **scopeParams** | scope | 查询范围 | 关联字段查询条件 |

#### 统计与显示
| 配置项 | 类型 | 描述 | 适用场景 |
|--------|------|------|----------|
| **fields** | field | 展示字段 | 元字段显示配置 |
| **showEl** | rule | 自定义内容 | 表达式显示内容 |
| **separator** | radio-input | 字段分隔符 | 多字段显示分隔符 |
| **showFieldName** | radio | 显示字段名 | 是否显示字段名称 |
| **recordSeparator** | radio-input | 记录分隔符 | 多条记录分隔方式 |

### 3.6 签名字段 (`signature`)
| 配置项 | 类型 | 描述 | 默认值 |
|--------|------|------|--------|
| **signatureStyle** | radio | 签名样式 | 弹窗(`dialog`)/直接显示(`show`) |

### 3.7 事件配置
| 配置项 | 类型 | 描述 | 功能 |
|--------|------|------|------|
| **events** | events | 自定义事件 | 字段级事件绑定 |

## 4. 使用流程示例

### 4.1 配置一个"部门选择"字段
1. **基础设置**：
   - `attriName`: "DEPT"
   - `attriLabel`: "所属部门"
   - `dataType`: "dept" (部门类型)
   - `isRequired`: "1" (必填)

2. **交互配置**：
   - `findPageId`: 选择部门选择页面
   - `objectScope`: 设置部门查询条件
   - `onlySelectLeaf`: false (可选中间节点)

3. **高级功能**：
   - `link`: true (启用链接)
   - `linkType`: "page" (链接到页面)
   - `linkPageId`: 选择部门详情页

4. **显示配置**：
   - `colspan`: 2 (占用2列)
   - `showLabel`: true (显示标签)

### 4.2 配置一个"金额输入"字段
1. **基础设置**：
   - `attriName`: "MONEY"
   - `attriLabel`: "订单金额"
   - `dataType`: "num" (数字类型)
   - `placeholder`: "请输入金额"

2. **验证配置**：
   - `isRequired`: "1"
   - `fieldExMethod`: "suffix" (后缀扩展)
   - `suffix`: "元"

3. **业务规则**：
   - `numericalUnits`: 从"数值单位"代码表选择
   - `showWay`: "value" (显示字段值)

4. **样式配置**：
   - `color`: "#f5222d" (红色)
   - `noteMode`: "tips" (弹窗说明)
   - `notes`: "请输入含税金额"
